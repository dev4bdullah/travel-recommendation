About Us

Travel Recommendation is a web app that helps users find amazing destinations worldwide.

Team members should follow these steps to maintain the project:

  • Update HTML pages in the `index.html`, `about.html`, and `contact.html` files.
  • Upload new images to the `images/` folder.
  • Push updates to GitHub and check the GitHub Pages live site.
